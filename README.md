# Setup 
## Quick Setup
Copy the config files in your home directory.
```
cp .vimrc ~/.vimrc
cp -r .config/nvim ~/.config/
```

Install plugins and LSP
```
nvim --headless +PlugInstall +TSUpdate +qa;
```
